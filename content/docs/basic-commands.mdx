This guide will help you understand and use the basic commands available on the server.

## Teleportation on the server
- `/spawn` - Teleports you to the spawn point of the server.
- `/home` - Teleports you to your set home location. Use `/sethome` to set your home.
- `/tpa <player>` - Sends a teleport request to another player.
- `/tpaccept` - Accepts a teleport request from another player.
- `/tpdeny` - Denies a teleport request from another player.
- `/msg <player> <message>` - Sends a private message to another player.

<Callout type="info">
	Actual limit for setting home is 1 per player. If you want to teleport to multiple locations, please use Waystones.
</Callout>

## Other commands
- `/slime` - Shows welcome screen with changelog information.
